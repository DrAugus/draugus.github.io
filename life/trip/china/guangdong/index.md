# 广东

## 广州

<LinkTravelogue city="广州" />

### 攻略

:::details 不愿透露姓名的群友推荐

广州特色，各类嫁女饼（叉烧酥，蛋黄酥，莲蓉酥等等），云吞面，白切鸡，烧鹅，拉肠，虾饺，烧麦，猪脚姜
> 嫁女饼是以前广州嫁女的时候，回礼给各位来宾的饼
广州还有各类凉茶，炖汤店，各类汤粉面

地铁二号线，市二宫站，同福中路开始打卡一直到同福东都能吃好几种
> “赞记”“安乐炖品”“芬芳甜品”“无名牛腩粉”“燊记面店”

市二宫地铁站D口出来，对面就有“赞记饼家”，买各类的老广烘焙饼，嫁女饼，糯米糍，酥皮面包等  
然后，再往前100米左右，回来对面马路，再到芬芳甜品，糖不甩和芒果糖水，还有煎粽子。芬芳的绿豆沙汤圆也好吃  
然后再往前100左右，对面就是“安乐炖品”几十年的老店，炖品店里的饭都很好吃，可以试试的  
同一方向往前，还有“无名”牛腩粉  
然后往前走~~同福东，有“燊记面店”牛三星，和猪油捞面，绝配，其余的面食都很可以  
然后，对面有熊记鸡粥，拉肠和粥都一绝

其实广州地道食品，都在老四区，东山，荔湾，越秀，海珠，其余的都是后期的，都是新派东西。找吃的，围着老四区就可以了

上下九有宝华面店，陈添记鱼皮，还有一家无名鱼蛋店，还有冰室，还有长寿面，还有多宝路的“冰之恋”只做冰镇

广州主要五类东西 甜品 粤菜 （早茶）烧味 啫啫煲 打边炉

千万别上广州塔，想去高处看夜景的话，可以去西塔楼顶的酒吧

## 珠海

<LinkTravelogue city="珠海" />

### 出行

:::tip 前往香港

喜欢坐船的可以在**九州港**码头，坐船去香港，建议直接到**港澳码头**，普通舱就可以，￥180，发船后如果窗边有空位可以去窗边看看的。船其实没有港珠澳大桥的大巴快的，大巴时速75左右，船的时速基本在60，激起的浪花比较大，玻璃基本都挂满了水。

推荐港珠澳大桥穿梭巴士，微信公众号**港珠澳大桥穿梭巴士**可以直接购买，你也可以看看实时汇率，看看人民币购买便宜还是港币购买便宜。
:::

:::tip 前往澳门
如果你在珠海站附近，步行即可，青茂口岸/拱北口岸都可，口岸外面的小店都会有寄存服务的，可以去问问
:::

## 梅州

<LinkTravelogue city="梅州" />

### 美食

<Food :foods="foodMZ" />

美食必吃  
腌面+三及第汤，一口面一口汤，满足感爆棚。还有客家猪/牛肉丸，Q弹有嚼劲。各类粄类（笋粄、艾粄、萝卜粄、发粄），丰顺捆粄，来梅州一定要尝尝梅州腌面，带着炸蒜酥的香气吃面时配上三及第汤，堪称绝配。来梅州一趟，当然不能错过腌面煮汤啦❗️
枸杞菜三及第是标配🍜
用猪油拌的腌面加上炸蒜蓉超级香！🔥
👉新宜家面馆，大埔腌面
随便找一家腌面店都不错👏

注意看看三及第是什么，可能不合口味。

:::details 尚未探索

围龙屋星园酒家
📍三角镇富奇路190号
由百年围龙屋打造的梅州特色餐厅，每个细节彰显着独特客家文化魅力，不管从屋顶屋角雕饰还是房间壁画都蕴含着浓郁客家文化元素，适合客家美食打卡和拍照📷
✨推荐：芝麻鸡，手磨豆腐，算盘子

1⃣️谢记鱼头煮粉
📍嘉应东路金良新村A栋103号
油炸过的鱼头放下去熬汤，熬出来的鱼汤奶白且鲜甜😋喜欢吃鱼的朋友不要错过❗️如果不想吃粉可以单点鱼头汤

2⃣️富奇路烧卖店 / 勤燕烧麦 / 虹桥烧麦
这几家都是本地人推荐的烧麦店❗️
用猪肉，笋干和香菇包裹的薄薄烧卖🥟
朋友可以选一家尝试一下😆

3⃣️金山牛杂（总店）
📍凌风西路16号
来梅州必吃之一❗️
跟广东其他地方吃到的牛杂是不同口感的，是冷吃牛杂，没有汤汁🥩值得朋友们来吃上这一口😍

4⃣️丰顺捆粄（百花洲店）/老邱捆粄
📍美凤路4-1号
一捆捆的皮薄馅足，各种口味都具备✌️
还有肠粉 炖汤随意搭配 谁不爱啊👍

📍三妹老屋新鲜焗盐鸡
💰人均45
👍🏻推荐菜：盐焗鸡，鸡杂米粉
最好提前45分钟打电话预约，现点现做要等很久

📍春香早餐店
💰人均10元
👍🏻推荐菜：三及第，猪腰汤，腌面，腌米粉
1份汤➕1份主食套餐总共10元，分量很足，这家的腌面和米粉口感偏硬，适合喜欢筋道口感的人。这家的内脏都处理得特别干净，没有一点异味。不喜欢内脏的也有纯瘦肉汤可选，吃得出材料都非常新鲜❗️

📍市塘唇萧屋烧麦
💰人均7元
👍🏻推荐菜：烧麦，腌水饺，三及第
本地居民推荐早餐店，烧卖2元/4颗，一口一颗的量，邻桌的食客都是每人三碟起点。这一家的三及第是放咸菜而不是枸杞叶。

📍大埔笋粄算盘子
💰人均2元
👍🏻推荐菜：忆子粄，鲜笋粄、笋干粄
位于老街的一家小吃店，各种大埔经典小吃，笋粄豆粄，均价2元，可以每种买一个尝尝，味道都不一样！

📍润心客家小吃
💰人均15
👍🏻推荐菜：味酵粄、牛肉丸、牛杂、算盘子
一定要点这家的牛肉丸，是用牛肉粒做的，和潮汕牛肉丸完全不同，非常特别‼️

📍松口家常菜馆
💰人均15
👍🏻推荐菜：番茄牛肉汤，猪肉丸汤，腌老鼠粄，炸丸子
就在这次我们住的酒店附近，番茄牛肉汤和猪肉丸汤都很鲜，炸丸子外酥里糯。

📍顾园饭店
💰人均60
👍🏻推荐菜：腌牛肉，蒸鲜腐竹，咸菜炒猪肠，炒米粉
家庭饭店，桌子不多，最好提前打电话订位，尤其推荐腌牛肉和蒸鲜腐竹，腐竹豆香味浓郁，在别的地方从没吃到过！

:::
