## 济州岛

济州岛的夏季潮湿多雨，多台风；冬季温差大，海风强烈，建议避开这两个季节前往，出行前请查询天气。济州岛时间比北京时间快1小时。

济州岛有三多：石头多、风多、女人多，这三多说明了济州岛作为一个海岛的特色。同时这里也被称为三无岛：无乞丐、无小偷、无大门。

:::tip 济州岛拥有亚热带海洋性气候，年平均气温16℃，一年四季都适合前往。

春季：3月-6月，温暖和煦，油菜花、樱花、绣球花纷纷绽放。  
夏季：6-8月，各大海滨浴场限时开放，是全年玩水的最好时段，也是全年最旺季，游客多、物价高一些。  
秋季：9月-11月，芒草盛开、枫叶渐红、柑橘丰收，是浪漫奇妙的季节。  
冬季：12月-2月，邂逅银装素裹的浪漫雪景、蒸汽腾腾的桑拿房，还有山茶花。
:::

### 交通

600路机场豪华大巴：往返于济州国际机场和南部中文旅游区、西归浦之间。上车点位于5号出口外，每20分钟一趟，上车购票，接受现金和Tmoney卡；价格在2000~5500韩元之间不等，根据距离而定。到济州市区约10分钟，到中文旅游区大约需要50分钟，到终点西归浦天堂酒店大约需要80分钟。行李箱可放在大巴车下层。

:::tip 济州岛公交分红、蓝、绿、黄4个不同类别。

🔴红色路线：快速公交。按距离收费，上车司机会问你目的地，费用2000-5500韩元间。停站少，会快一些。  
🔵🟢蓝色和绿色：连接市区和郊区的日常公交。费用统一1150韩元随便坐，但站多、耗时久、班次相对多一些（10-40分钟不等）。日常在市区活动、去景点也经常会搭乘这两类公交。  
🟡黄色线路：循环观光巴士。主要经过济州岛内一些比较重要的景点，如民俗五日市场、天地渊瀑布、神话历史公园、榧子林。费用统一1150韩元，班次少（0.5-1小时一趟）。

:::

:::warning 交通提醒
1、所有公交车都接受刷Tmoney卡和现金，但现金收费略高，如刷Tmoney卡1150韩元，现金则要收取1200韩元，向车内投钱后会自动找硬币。
2、很多公交车需要等待20分钟以上，建议提前在Naver上看发车时间，临近时间再去车站等候。
3、奶茶、咖啡等开口饮品不允许被带上公交车，否则很有可能被司机拒载。
4、临近下车点，需要自行按车铃示意司机停车。
:::

## **韩国旅游计划**  

以下是针对韩国旅游的行程建议，结合你已有日本旅行经验（文化相似但各有特色），从时间、景点、美食等方面为你规划：  
  
一、行程时长建议  

1. **3天行程**：适合首尔核心区快节奏打卡，适合时间紧张或商务出差顺带游览。  
2. **5天行程**：更推荐！可深度体验首尔+近郊（如仁川、水原），节奏舒适。  
3. **若超过5天**：可加入釜山、济州岛等异地行程（需飞机/高铁，建议单独规划）。  
  
二、行程安排参考（以**5天为例）**  
:::info **Day 1：首尔传统与现代融合**  

- **上午**：景福宫（守门将换岗仪式）+ 北村韩屋村（传统建筑拍照）。  
- **下午**：三清洞艺术街区→仁寺洞（手工艺品、茶屋）。  
- **晚上**：明洞夜市（小吃+购物），推荐辣炒年糕、蜂蜜黄油炸鸡。
:::  
:::info **Day 2：潮流文化与城市全景**  
- **上午**：梨花女子大学周边街区（文艺小店）+ 弘大（街头表演、潮牌）。  
- **下午**：延南洞咖啡厅休憩→汉江公园（骑行或野餐）。  
- **晚上**：南山首尔塔夜景（缆车上山）。  
:::
:::info **Day 3：历史与本地生活**  
- **全天**：釜山（可选，乘KTX约2.5小时）或近郊水原华城（ UNESCO遗产）。  
- **替代方案**：首尔江东区 COEX Mall（星空图书馆）+ 乐天世界塔观景。  
:::
:::info **Day 4：韩流体验与购物**  
- **上午**：SM Town博物馆（K-POP粉丝必去）或国立中央博物馆。  
- **下午**：新沙洞林荫道（设计师品牌）+ 狎鸥亭罗德奥（奢侈品）。  
- **晚上**：观看乱打秀（非语言喜剧，跨文化易懂）。  
:::
:::info **Day 5：放松与返程**  
- **上午**：广藏市场（绿豆煎饼、生拌牛肉）体验市井烟火气。  
- **下午**：汗蒸幕（如龙山汗蒸幕）缓解疲劳，准备返程。  
:::

三、**3天精简版行程**  

- **Day1**：景福宫→北村韩屋→明洞夜市  
- **Day2**：弘大→汉江公园→南山塔  
- **Day3**：广藏市场→COEX Mall→仁寺洞  
  
四、实用贴士（对比日本经验）  

1. **交通**：类似日本IC卡，购买T-money卡（地铁、巴士、便利店通用）。  
2. **语言**： tourist区域英语标识较多，但基础韩语问候（如“감사합니다谢谢”）更便利。  
3. **支付**：支付宝/微信支付在明洞、弘大等地普及，但备少量现金。  
4. **文化差异**：韩国更注重辈分礼仪（如双手接物），但整体比日本随意。  
  
五、推荐季节  

- **春秋（4-5月、9-10月）**：气候宜人，樱花/枫叶季媲美日本。  
- 避开6-8月梅雨季和1月极寒期。

## **与中国、日本市场的对比总结**  

| 特征    | 中国          | 日本                   | 韩国                   |
| ----- | ----------- | -------------------- | -------------------- |
| 市场结构  | 高度集中，巨头垄断   | 高度分散，专业连锁林立          | 相对集中，财阀主导高端，中端和经济型分散 |
| 品牌特色  | 按价格和档次划分清晰  | 特色服务导向（浴场、早餐）        | 设计与奢华体验导向非常突出        |
| 代表性品牌 | 华住会、锦江、首旅如家 | 东横INN、Dormy Inn、三井花园 | 乐天、新罗、GLAD、L’Escape  |
| 独特业态  | 相对较少        | 温泉旅馆、胶囊旅馆            | 汗蒸幕过夜、韩屋体验           |
  
简单来说：  

- **中国**是“规模为王”的巨人战场。  
- **日本**是“细节服务”的百货商场。  
- **韩国**是“顶奢与设计”的精品店，同时拥有独特的本地文化体验。  
希望这份补充能让你对东亚三国的酒店格局有更完整的了解！

**核心差异总结**  

1. **市场结构**：中国是“巨头垄断”，几家大集团通过收购覆盖全市场。日本是“诸侯割据”，虽有大型集团，但众多专业的商务酒店连锁品牌并存，市场更细分。  
2. **品牌逻辑**：中国品牌主要按**价格和档次**划分。日本品牌在按档次划分的同时，**特色服务**（如大浴场、免费拉面/早餐）是极其重要的区分维度。  
3. **特色业态**：日本拥有中国几乎不存在的**温泉旅馆**和**胶囊旅馆**等独特业态。
