# PS 技巧

## ps如何等比例裁剪

方法一：快捷键快速等比例裁剪（临时比例）  
打开图片，按 C 激活裁剪工具，图像四周出现裁剪框。  
按住 Shift 键拖动裁剪框的对角控制点，裁剪区域会按原始比例等比缩放；同时按住 Shift+Alt，可从中心向四周等比缩放。  
调整好保留区域后，按 Enter 键或双击裁剪框内确认裁剪。

方法二：预设 / 自定义固定比例裁剪（精准比例）  
激活裁剪工具（C），在顶部选项栏找到 “比例” 下拉菜单。  
选择内置比例（如 1:1、4:3、16:9 等），或选择 “比例” 后手动输入自定义宽高比（如 3:2）。  
拖动裁剪框或控制点，框会强制保持设定比例，仅调整位置与大小。  
确认后按 Enter 或点击选项栏的 ✅ 完成裁剪。

方法三：固定尺寸 + 比例裁剪（精确尺寸）  
激活裁剪工具，在选项栏输入固定的宽、高、分辨率（如 800px 宽、600px 高、72dpi）。  
拖动裁剪框选取保留区域，系统会自动按设定比例与尺寸裁剪。  
按 Enter 或双击完成，裁剪后图像尺寸与分辨率即符合输入值。
